# Dual-View Chat 

---
### How to start the app
  ### Backend:
  ```
  *in ./dual-chat-backend
      1. add .env file
      2. npm install
      3. npm run start:dev
  ```
    
  ### Frontend:
  ```
    *in ./dual-chat-frontend
      1. add .env file
      2. npm install
      3. npm run dev
  ```
